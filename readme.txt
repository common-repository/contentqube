Import ContentQube feeds into your WordPress CMS

== Description ==

The plugin integrates with OneQube's ContentQube platform.
It allows the user to quickly curate, publish and syndicate ContentQube content to their WordPress blog.

For more information visit: [OneQube.com](https://oneqube.com/softwareservices/contentqube/)

To use the plugin we **require** you to have current [ContentQube Subscription](https://oneqube.com/softwareservices/contentqube/)

== Installation ==
1. Install the plugin and activate it.
1. Navigate to the plugin settings and enter your ContentQube account information.
1. Map the posts to your WordPress categories.

Find a complete example [here](https://oneqube.com/2017/04/19/setting-contentqube-plugin/).

== Frequently Asked Questions ==
= Do I need a ContentQube account to use this plugin =
Yes, a current ContentQube subscription is required.
= Is my own content automatically shared in your network =
No. For more information read our [privacy policy](http://contentqubenetwork.com/privacy-policy/).
